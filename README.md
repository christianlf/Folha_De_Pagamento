# Folha_De_Pagamento

Sistema de Gerenciamento de Funcionários:

Este repositório contém um sistema de gerenciamento de funcionários escrito em Python. O sistema permite inserir, remover, atualizar informações e calcular folhas de pagamento de funcionários. Ele é ideal para pequenas e médias empresas que precisam de uma maneira eficiente de gerenciar seus funcionários.

Funcionalidades:

Inserir Funcionário: Adicione novos funcionários com informações como matrícula, nome, código de função, faltas, salário bruto e volume de vendas.
Remover Funcionário: Remova funcionários usando sua matrícula.
Determinar Folha de Pagamento: Calcule e exiba a folha de pagamento de um funcionário específico, incluindo faltas, comissões e impostos.
Gerar Relatório: Gere um relatório com informações básicas de todos os funcionários.
Adicionar Faltas: Adicione faltas ao registro de um funcionário.
Atualizar Vendas: Atualize o volume de vendas de um vendedor.
Maior Salário Líquido: Encontre e exiba o funcionário com o maior salário líquido.
Maior Número de Faltas: Encontre e exiba o funcionário com o maior número de faltas.

Estrutura do Código:

verificação: Valida a entrada do usuário para garantir que é do tipo correto.
inserir_funcionario: Insere um novo funcionário no dicionário funcionarios.
remover_funcionario: Remove um funcionário do dicionário funcionarios com base na matrícula fornecida.
determinar_folha_pagamento: Calcula e exibe a folha de pagamento de um funcionário específico.
gerar_relatorio: Gera um relatório com informações básicas de todos os funcionários.
calcular_salario_liquido: Calcula o salário líquido de um funcionário após deduções de faltas e impostos, e adição de comissões.
maior_salario_liquido: Encontra e exibe o funcionário com o maior salário líquido.
maior_faltas: Encontra e exibe o funcionário com o maior número de faltas.
Adicionar_faltas: Adiciona uma falta ao registro de um funcionário.
Atualizar_Vendas: Atualiza o volume de vendas de um vendedor.
menu: Exibe um menu para interagir com o sistema e chamar as funções apropriadas com base na escolha do usuário.
